# Student Record Management System – Console & Web Based Application

It is a modern and efficient student data management application built using **C++, File Handling, Socket Programming, HTML, CSS, and JavaScript**. The project supports both **Console Mode** and **Web Server Mode**, allowing users to manage student records through a terminal-based interface as well as a browser-based dashboard.

## Features

- 🎓 Complete student record management system  
- 💻 Dual-mode support: Console Interface + Web Interface  
- 🌐 Built-in web server using C++ socket programming  
- 📂 File handling support for permanent data storage  
- ➕ Add, update, delete, and search student records  
- 📊 Automatic percentage and grade calculation  
- 🔍 Filter students by grades and performance  
- 📈 Sort student records based on percentage  
- 🔌 REST-style API endpoints for frontend communication  
- ⚡ Fast and lightweight architecture using native C++

## Core Functionalities

- **Console Mode** – Text-based interface for managing student records  
- **Web Server Mode** – Browser-based interface powered by a custom C++ server  
- **Student CRUD Operations** – Create, read, update, and delete records  
- **Grade & Percentage Calculation** – Automatic result processing system  
- **Persistent Storage** – Saves and loads student records using text files  
- **Sorting & Filtering** – Organizes students based on percentage and grades  
- **JSON Response System** – Converts records into JSON format for APIs  
- **HTTP Request Handling** – Supports GET, POST, PUT, DELETE, and OPTIONS methods  

## API Endpoints

- **/api/students** – Fetch all student records  
- **/api/students/{roll}** – Fetch a specific student record  
- **POST /api/students** – Add new student data  
- **PUT /api/students/{roll}** – Update existing student record  
- **DELETE /api/students/{roll}** – Delete a student record  
- **/api/sort** – Sort student records by percentage
